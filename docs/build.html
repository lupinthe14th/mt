Build¶

OpenNMT の Lua 実装のDockerバージョンを利用する。

Install¶

Docker¶

Dockerhubのイメージは、　Dockerfileを比べる限り SentencePiece のインストール が含まれていないため、github リポジトリのDockerfileを利用して、Dockerイメージ をBuildする。

  1. OpenNMT リポジトリのクローン

    $ cd ${repo_root}
$ git clone https://github.com/OpenNMT/OpenNMT.git

  2. Docker build

    $ cd OpenNMT
$ sudo docker build . -t opennmt/opennmt:latest

    注釈

    私の利用した環境 [1] では、5時間ほどかかった

Configuration¶

Docker Compose¶

サービス単位で構成して簡単に使いたいため、Docker Composeを利用する。

Docker Compose構成ファイル¶
---
version: "3.6"
services:
  train:
    image: opennmt/opennmt:latest
    volumes:
      - ./data:/data

  rest:
    image: opennmt/opennmt:latest
    volumes:
      - ./data:/data
    command: th tools/rest_translation_server.lua -config /data/rest.yml
    ports:
      - 7784:7784

  0mq:
    image: opennmt/opennmt:latest
    volumes:
      - ./data:/data
    command: th tools/translation_server.lua -config /data/0mq.yml
    ports:
      - 5556:5556
